# Face Recognition & Identity Verification System
## 🛡️ L'alliance de la Vision par Ordinateur et de la Cybersécurité

À une époque où l'identité numérique est au cœur des préoccupations mondiales, ce projet propose une réponse technique à la fois accessible et performante. En tant qu'étudiant en cybersécurité, mon ambition était de concevoir un système de reconnaissance faciale qui dépasse le stade du simple prototype pour devenir un véritable outil de vérification d'identité, capable de s'adapter aux exigences de l'automatisation moderne.

### 📌 Présentation du Projet
Cette application, développée entièrement en **Python**, utilise la puissance de la bibliothèque **OpenCV** pour transformer une capture d'image brute en un processus d'authentification fiable. Elle identifie des individus à partir d'un flux vidéo en direct ou d'images statiques en les comparant à une base de données d'entraînement prédéfinie.

### 🚀 Fonctionnalités Clés
*   **Reconnaissance en Temps Réel** : Analyse instantanée via webcam pour une vérification immédiate.
*   **Interface Intuitive (GUI)** : Utilisation de **Tkinter** pour offrir une expérience utilisateur fluide, masquant la complexité technique derrière un tableau de bord ergonomique.
*   **Traitement de Données Optimisé** : Intégration de **NumPy** pour une gestion rapide et efficace des matrices d'images.
*   **Architecture Structurée** : Organisation des profils dans le répertoire `training-data/` pour faciliter l'ajout de nouveaux visages.

### 🛠 Stack Technique
*   **Langage** : Python
*   **Bibliothèques** : 
    *   `opencv-contrib-python >= 4.5.0`
    *   `numpy >= 1.21.0`
    *   `Tkinter`
